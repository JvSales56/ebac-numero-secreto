# Jogo da Adivinhação

Um simples jogo da adivinhação desenvolvido com HTML, CSS e JavaScript.

Projeto desenvolvido durante o curso de Front-end da EBAC.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Geração aleatória de um número secreto entre 1 e 100
- Inserção manual de palpites
- Geração automática de um palpite aleatório
- Dicas indicando se o número secreto é maior ou menor
- Contador de tentativas
- Encerramento do jogo após 10 tentativas
- Mensagens de vitória e derrota

## Como utilizar

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Clique em **Começar Jogo** para gerar um número secreto.
4. Digite um número entre **1** e **100** ou utilize o botão **Chutar** para gerar um palpite aleatório.
5. Tente descobrir o número secreto em até **10 tentativas**.
